<p>오늘 CSS 과제로 뮤직 플레이어 UI를 직접 만들어봤다. 배운 개념들을 실제로 적용해보니까 훨씬 머릿속에 잘 들어오는 느낌이었다. 처음에는 교수님께서 보여주셨던 예시와 테두리 색까지 정말 똑같이 만들어서, &quot;센스가 없었다&quot;는 웃지 못한 상황도 발생했지만, 이후에 나름대로의 스타일을 주어서 꾸며보았다.</p>
<p><img alt="" src="https://velog.velcdn.com/images/addung/post/4d7203fb-1086-4ea4-966e-9284ec9102fd/image.png" /></p>

<h2 id="1-전체-구조">1. 전체 구조</h2>
<p>HTML 구조는 심플하게 두 덩어리로 나눴다.</p>
<pre><code>music-player (section)
├── photo (div) → 앨범 커버 이미지
└── info-zone (div) → 제목 + 가사 + 오디오 플레이어</code></pre><pre><code class="language-html">&lt;section class=&quot;music-player&quot;&gt;
    &lt;div class=&quot;photo&quot;&gt;
        &lt;img src=&quot;nell.jpg&quot; alt=&quot;앨범커버&quot;&gt;

    &lt;div class=&quot;info-zone&quot;&gt;
        &lt;p class=&quot;title&quot;&gt;기억을 걷는 시간 - 넬(NELL)&lt;/p&gt;
        &lt;p class=&quot;lyrics&quot;&gt;오늘도 난 너의 흔적 안에 살았죠&lt;/p&gt;

    &lt;/div&gt;

<hr />
<h2 id="2-css-코드-분석">2. CSS 코드 분석</h2>
<h3 id="가-body--화면-정중앙-배치">가. body — 화면 정중앙 배치</h3>
<pre><code class="language-css">body {
    margin: 0;
    display: flex;
    justify-content: center;
    align-items: center;
    min-height: 100vh;
    background-color: blanchedalmond;
}</code></pre>
<ul>
<li><code>display: flex</code> + <code>justify-content: center</code> + <code>align-items: center</code> : 수평/수직 모두 중앙 정렬</li>
<li><code>min-height: 100vh</code> : 뷰포트 전체 높이를 최소 높이로 설정 → 화면이 작아도 중앙 유지</li>
<li><code>margin: 0</code> : 브라우저 기본 여백 제거</li>
</ul>
<blockquote>
<p><strong>포인트:</strong> flex로 body를 잡으면 자식 요소 하나를 화면 정중앙에 놓는 게 엄청 쉬워진다.</p>
</blockquote>
<hr />
<h3 id="나-music-player--카드-모양-만들기">나. .music-player — 카드 모양 만들기</h3>
<pre><code class="language-css">.music-player {
    display: flex;
    justify-content: center;
    align-items: center;
    border: 3px solid yellowgreen;
    box-shadow: 10px 10px 10px rgba(0,0,0,0.3);
    border-radius: 20px;
}</code></pre>
<ul>
<li><code>display: flex</code> : 안쪽 photo와 info-zone을 가로로 나란히 배치</li>
<li><code>border: 3px solid yellowgreen</code> : 연두색 테두리</li>
<li><code>box-shadow: 10px 10px 10px rgba(0,0,0,0.3)</code> : 오른쪽 아래로 살짝 그림자 효과</li>
<li><code>border-radius: 20px</code> : 모서리 둥글게</li>
</ul>
<blockquote>
<p><strong>box-shadow 값 순서:</strong> <code>x축 y축 흐림정도 색상</code></p>
<ul>
<li>x, y 값이 클수록 그림자가 멀리 떨어짐</li>
<li>rgba의 마지막 숫자(0.3)는 투명도 → 낮을수록 연해짐</li>
</ul>
</blockquote>
<hr />
<h3 id="다-photo-photo-img--앨범-이미지">다. .photo, .photo img — 앨범 이미지</h3>
<pre><code class="language-css">.photo {
    margin: 10px;
    padding: 10px;
}

.photo img {
    width: 170px;
    height: auto;
}</code></pre>
<ul>
<li><code>height: auto</code> : width에 맞게 높이가 자동으로 비율 유지됨</li>
<li>이미지 크기를 width만 고정하고 height는 auto로 놓으면 이미지 비율이 안 깨진다.</li>
</ul>
<hr />
<h3 id="라-info-zone--텍스트--오디오-영역">라. .info-zone — 텍스트 + 오디오 영역</h3>
<pre><code class="language-css">.info-zone {
    width: 600px;
    margin: 0 10px 0 0;
}

.title {
    color: green;
    font-size: 30px;
    padding: 10px;
    margin: 10px 10px 0 2px;
}

.lyrics {
    margin: 5px 10px 5px 2px;
}

audio {
    width: 580px;
    margin: 0 0 0 2px;
}</code></pre>
<ul>
<li><code>margin: 0 10px 0 0</code> : 오른쪽에만 10px 여백 (상 우 하 좌 순서)</li>
<li><code>audio</code>에 width를 직접 줘서 플레이어 크기 고정</li>
<li><code>font-size: 30px</code> 으로 곡 제목을 크게 강조</li>
</ul>
<hr />
<h2 id="3-오늘-과제하면서-배운-것">3. 오늘 과제하면서 배운 것</h2>
<p><strong>1. flex는 진짜 강력하다</strong></p>
<p>body에 flex 세 줄만 써도 화면 정중앙 배치가 끝났다. 예전에 <code>margin: auto</code> 로 삽질하던 것과 비교하면 훨씬 편하다.</p>
<p><strong>2. box-shadow로 입체감 주기</strong></p>
<p>테두리만 있을 때보다 그림자 하나 추가했을 때 훨씬 카드처럼 보인다. rgba의 투명도 값으로 그림자 강도를 조절하는 게 포인트.</p>
<p><strong>3. height: auto 습관 들이기</strong></p>
<p>이미지에 width만 고정하고 height는 auto로 놓으면 비율이 안 깨진다. 처음에 height도 px로 고정했다가 이미지가 찌그러지는 걸 봤다.</p>
<p><strong>4. margin 방향 순서</strong></p>
<p><code>margin: 상 우 하 좌</code> — 시계 방향이라는 걸 이번에 확실히 외웠다.</p>
<hr />
<h2 id="오늘-과제-마무리">오늘 과제 마무리</h2>
<p>단순한 뮤직 플레이어지만 flex, border-radius, box-shadow, audio 태그까지 배운 걸 다 활용해볼 수 있었다. 실제로 만들어보니까 어떤 속성이 어디에 영향을 주는지 훨씬 명확하게 보였다. 다음엔 hover 효과도 넣어보고 지금까지 배웠던 이모저모의 다양한 기능을 넣어봐야 겠다 싶었다.
여담이지만 &quot;display: flex + justify-content: center + align-items: center : 수평/수직 모두 중앙 정렬&quot; 이 세가지를 이해하는게 조금 어려웠다. 이 세가지를 다양하게 적용하며 익혀봐야겠다! </p>
