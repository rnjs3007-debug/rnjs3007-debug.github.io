<h2 id="1-반복문-제어--break와-continue">1. 반복문 제어 — <code>break</code>와 <code>continue</code></h2>
<h3 id="break">break</h3>
<p>반복문 내에서 <code>break</code>를 만나면 <strong>현재 반복문 하나를 즉시 탈출</strong>한다.<br />중첩 반복문에서는 가장 안쪽 반복문 하나만 탈출한다는 점에 주의해야한다.</p>
<pre><code class="language-c">for (int i = 0; i &lt; 10; i++) {
    if (i == 5) break;  // i가 5가 되면 반복문 탈출
    printf(&quot;%d\n&quot;, i);
}
// 출력: 0 1 2 3 4</code></pre>
<h3 id="continue">continue</h3>
<p>특정 조건의 반복 <strong>회차만 건너뛰고</strong>, 반복문은 계속 진행한다.</p>
<pre><code class="language-c">for (int i = 0; i &lt; 10; i++) {
    if (i % 2 == 0) continue;  // 짝수면 건너뜀
    printf(&quot;%d\n&quot;, i);
}
// 출력: 1 3 5 7 9</code></pre>
<blockquote>
<p>💡 <strong>정리</strong>: <code>break</code>는 반복문 탈출, <code>continue</code>는 해당 회차만 스킵!</p>
</blockquote>

<h2 id="2-수업-과제-풀이">2. 수업 과제 풀이</h2>
<h3 id="문제-1--약수-약수-개수-약수-합-구하기">문제 1 — 약수, 약수 개수, 약수 합 구하기</h3>
<p>양의 정수를 입력받아 약수, 개수, 합을 출력한다.</p>
<pre><code class="language-c">#include &lt;stdio.h&gt;
#pragma warning(disable:4996)

int main(void) {
    int i = 1;
    int num;
    int count = 0;
    int sum = 0;

    printf(&quot;input number: &quot;);
    scanf(&quot;%d&quot;, &amp;num);

    if (num &lt;= 0) {
        printf(&quot;양의 정수를 입력해주세요!&quot;);
        exit(1);  // 잘못된 입력 시 종료
    }

    printf(&quot;약수: &quot;);
    for (i = 1; i &lt;= num; i++) {
        if (num % i == 0) {
            printf(&quot;%d &quot;, i);
            count++;
            sum += i;
        }
    }

    printf(&quot;\n개수: %d\n&quot;, count);
    printf(&quot;합: %d\n&quot;, sum);

    return 0;
}</code></pre>
<p><strong>실행 예시 (입력: 12)</strong></p>
<pre><code>약수: 1 2 3 4 6 12
개수: 6
합: 28</code></pre><hr />
<h3 id="문제-2--짝수이면서-3의-배수인-숫자들의-합구하기">문제 2 — 짝수이면서 3의 배수인 숫자들의 합구하기</h3>
<pre><code class="language-c">#include &lt;stdio.h&gt;
#pragma warning(disable:4996)

int main(void) {
    int i, num;
    int sum = 0;

    printf(&quot;input number: &quot;);
    scanf(&quot;%d&quot;, &amp;num);

    if (num &lt;= 1) {
        printf(&quot;1보다 큰 수를 입력해주세요!&quot;);
        exit(1); #오류 막기위해 항상 입력!
    }

    for (i = 1; i &lt;= num; i++) {
        if (i % 2 == 0 &amp;&amp; i % 3 == 0) {
            sum += i;
        }
    }

    printf(&quot;sum = %d\n&quot;, sum);
    return 0;
}</code></pre>

<h3 id="문제-3--각-자리수의-합이-10인-숫자-출력">문제 3 — 각 자리수의 합이 10인 숫자 출력</h3>
<p>핵심 아이디어: 숫자를 10으로 계속 나누면서 자리수를 하나씩 분리한다.</p>
<pre><code class="language-c">#include &lt;stdio.h&gt;
#pragma warning(disable:4996)

int main(void) {
    int num;
    printf(&quot;input number: &quot;);
    scanf(&quot;%d&quot;, &amp;num);

    if (num &lt;= 1) {
        printf(&quot;1보다 큰 수를 입력해주세요!&quot;);
        exit(1);
    }

    for (int i = 1; i &lt;= num; i++) {
        int sum = 0;
        int answer = i;  // i를 복사해서 사용 (i 자체를 변경하면 안 됨)

        while (answer &gt; 0) {
            sum += answer % 10;  // 일의 자리 추출
            answer /= 10;        // 숫자를 한 자리씩 줄임
        }

        if (sum == 10) {
            printf(&quot;%d\n&quot;, i);
        }
    }

    return 0;
}</code></pre>
<blockquote>
<p>💡 <strong>자리수 분리 공식 정리</strong>
| 자리 | 공식 |
| 일의 자리 | <code>num % 10</code> |
| 십의 자리 | <code>(num % 100) / 10</code> |
| 백의 자리 | <code>(num % 1000) / 100</code> |
%10과 /10을 계속해서 반복하면 구할 수 있다.</p>
</blockquote>

<h3 id="문제-4--각-자리수-중-가장-큰-숫자-출력">문제 4 — 각 자리수 중 가장 큰 숫자 출력</h3>
<pre><code class="language-c">#include &lt;stdio.h&gt;
#pragma warning(disable:4996)

int main(void) {
    int num;
    int max = -1;
    int digit;

    printf(&quot;input number: &quot;);
    scanf(&quot;%d&quot;, &amp;num);

    while (num &gt; 0) {
        digit = num % 10;       // 일의 자리 추출
        if (digit &gt; max) 
        max = digit;
        num /= 10;              // 자리수 줄이기
    }

    printf(&quot;max = %d\n&quot;, max);
    return 0;
}</code></pre>
<blockquote>
<p>💡 <code>max</code>의 초기값은 <code>-1</code>로 설정하는 게 더 안전하다. (모든 자리수가 0일 때도 대응 가능함)</p>
</blockquote>

<h2 id="3-1차원-배열">3. 1차원 배열</h2>
<h3 id="배열이-필요한-이유">배열이 필요한 이유</h3>
<p>변수가 많아질수록 선언이 번거로워진다.</p>
<pre><code class="language-c">// 배열 없이
int num0, num1, num2, ..., num29;  // 30개 선언 

// 배열 사용
int num[30];  // 한 줄로 해결!</code></pre>
<h3 id="배열-선언-규칙">배열 선언 규칙</h3>
<pre><code class="language-c">// 올바른 선언
int arr[30];         // O

#define SIZE 30
int arr[SIZE];       // O — #define 상수는 가능

// 잘못된 선언
int idx = 30;
int arr[idx];        // X — 일반 변수는 불가

const int idx = 30;
int arr[idx];        // X — const도 불가 (C 기준)</code></pre>
<blockquote>
<p>💡 배열 크기에는 반드시 <strong>상수 또는 <code>#define</code> 매크로</strong>를 사용해야 한다.</p>
</blockquote>
<h3 id="배열-기본-사용법">배열 기본 사용법</h3>
<pre><code class="language-c">int arr[5];

// 값 넣기
arr[0] = 10;
arr[1] = 20;

// 반복문으로 입력받기
for (int i = 0; i &lt; 5; i++) {
    scanf(&quot;%d&quot;, &amp;arr[i]);
}

// 반복문으로 출력하기
for (int i = 0; i &lt; 5; i++) {
    printf(&quot;arr[%d] = %d\n&quot;, i, arr[i]);
}</code></pre>
<blockquote>
<p>⚠️ <strong>인덱스는 0부터 시작</strong>함에, <code>arr[5]</code>로 선언하면 인덱스는 <code>0 ~ 4</code>까지이다.</p>
</blockquote>

<h2 id="4-문자-배열과-문자열">4. 문자 배열과 문자열</h2>
<h3 id="문자char-vs-문자열string">문자(char) vs 문자열(string)</h3>
<pre><code class="language-c">char a = 'a';          // 문자 하나 — 작은따옴표
char arr[10] = &quot;a&quot;;    // 문자열 — 큰따옴표, 끝에 '\0' 자동 추가</code></pre>
<h3 id="널null-문자-0">널(null) 문자 <code>'\0'</code></h3>
<p>문자열의 끝을 알리는 특수 문자이다. 배열 선언 시 <strong>반드시 널값이 들어갈 공간 +1</strong>을 해줘야 한다.</p>
<pre><code class="language-c">char ch[6] = &quot;hello&quot;;
// ch[0]='h', ch[1]='e', ch[2]='l', ch[3]='l', ch[4]='o', ch[5]='\0'</code></pre>
<h3 id="출력-방법">출력 방법</h3>
<pre><code class="language-c">// 문자 하나 출력
printf(&quot;%c&quot;, ch[0]);

// 문자열 전체 출력
printf(&quot;%s&quot;, ch);</code></pre>
<h3 id="문자열-입력-받기">문자열 입력 받기</h3>
<pre><code class="language-c">char ch[100];
scanf(&quot;%s&quot;, ch);  // 문자열은 &amp; 없이 그냥 이름만 씀!</code></pre>

<h2 id="5-컴퓨터가-문자를-저장하는-방법--아스키-코드ascii">5. 컴퓨터가 문자를 저장하는 방법 — 아스키 코드(ASCII)</h2>
<h3 id="1의-두-가지-의미">'1'의 두 가지 의미</h3>
<p>컴퓨터는 모든 것을 0과 1(이진수)로만 저장한다. 그래서 우리가 키보드로 <code>1</code>을 눌렀을 때, 이것이 <strong>계산용 숫자</strong>인지 <strong>화면에 보여줄 글자</strong>인지 구분해야만 한다</p>
<table>
<thead>
<tr>
<th>종류</th>
<th>타입</th>
<th>의미</th>
</tr>
</thead>
<tbody><tr>
<td>숫자 데이터</td>
<td><code>int</code></td>
<td>진짜 크기를 가진 수. 산술 연산(<code>1 + 1 = 2</code>) 용도</td>
</tr>
<tr>
<td>문자 데이터</td>
<td><code>char</code></td>
<td>화면에 그리기 위한 '기호'. 고유 번호표(아스키 코드)로 저장</td>
</tr>
</tbody></table>
<h3 id="아스키-코드란">아스키 코드란?</h3>
<p>컴퓨터가 글자를 저장할 때 사용하는 <strong>고유 번호표</strong>이다.<br /><code>%s</code>나 <code>%c</code>로 입력을 받으면 숫자가 아니라 <strong>글자(기호)</strong> 로 해석하기 때문에, 아스키 코드 번호가 메모리에 저장된다.</p>
<p>사용자가 키보드로 <code>1</code>을 누르면 → 컴퓨터는 <code>'1'</code>이라는 모양의 글자로 인식 → 메모리에 <strong>아스키 코드 49</strong>를 저장한다.</p>
<h3 id="왜---0을-하면-숫자가-될까">왜 <code>- '0'</code>을 하면 숫자가 될까?</h3>
<p>아스키 코드 설계자들이 숫자 글자들을 <strong>순서대로 배치</strong>해뒀기 때문이다.</p>
<pre><code>'0' = 48번
'1' = 49번
'2' = 50번
 ...
'9' = 57번</code></pre><p>그래서 어떤 숫자 문자에서 <code>'0'</code>(48)을 빼면 실제 숫자 값이 나온다.</p>
<pre><code class="language-c">// 문자 '3'을 숫자 3으로 변환하기
char c = '3';           // 메모리에는 51이 저장됨
int num = c - '0';      // 51 - 48 = 3
printf(&quot;%d\n&quot;, num);    // 출력: 3</code></pre>
<blockquote>
<p>💡 <strong>비유로 이해하기</strong></p>
<ul>
<li><code>int</code>로 입력받기 → &quot;사과 <strong>1개</strong>&quot;라고 말하면 컴퓨터가 바로 <strong>수량</strong>으로 이해</li>
<li><code>char</code>로 입력받기 → &quot;1&quot;이라는 글자를 종이에 쓰는 것. 종이의 '1'은 그 자체로 수량이 아닌 그냥 <strong>그림</strong>. <code>- '0'</code>은 그 그림을 다시 수량으로 해석하는 과정이라고 생각하자</li>
</ul>
</blockquote>
<h3 id="정리">정리</h3>
<ol>
<li>컴퓨터는 글자를 저장할 때 <strong>고유 번호(아스키 코드)</strong> 를 사용한다.</li>
<li><code>%s</code> / <code>%c</code>로 입력받으면 숫자가 아닌 <strong>글자</strong>로 취급되어 번호가 저장된다.</li>
<li>숫자 글자의 번호는 <code>'0'</code>~<code>'9'</code>가 순서대로 매겨져 있어, <strong><code>- '0'</code>(또는 <code>- 48</code>)</strong> 을 하면 실제 값을 구할 수 있다.</li>
</ol>

<h2 id="6-배열-응용-문제-풀이">6. 배열 응용 문제 풀이</h2>
<h3 id="1-회문palindrome-판별">1) 회문(Palindrome) 판별</h3>
<p>앞뒤가 똑같은 문자열인지 확인하는 프로그램.</p>
<pre><code class="language-c">#include &lt;stdio.h&gt;
#pragma warning(disable:4996)

#define TRUE 1
#define FALSE 0

int main(void) {
    int len = 0;
    char ch[100];
    int i;
    int flag = TRUE;

    scanf(&quot;%s&quot;, ch);

    // 문자열 길이 구하기
    for (len = 0; ch[len] != '\0'; len++);

    // 앞과 뒤를 비교
    for (i = 0; i &lt; len / 2; i++) {
        if (ch[i] != ch[len - 1 - i]) {
            flag = FALSE;
            break;
        }
    }

    if (flag == TRUE)
        printf(&quot;it's palindrome\n&quot;);
    else
        printf(&quot;it's not palindrome\n&quot;);

    return 0;
}</code></pre>
<h3 id="배열-뒤집기-swap">배열 뒤집기 (swap)</h3>
<p>swap 기법으로 배열 원소의 위치를 교환한다.</p>
<pre><code class="language-c">int nums[5] = {1, 2, 3, 4, 5};

for (int i = 0; i &lt; 5 / 2; i++) {
    int temp = nums[i];
    nums[i] = nums[4 - i];
    nums[4 - i] = temp;
}
// 결과: 5 4 3 2 1</code></pre>
<blockquote>
<p>💡 <strong>swap의 핵심</strong>: 임시 변수 <code>temp</code>를 이용해 값을 교환한다.</p>
<pre><code>temp = a;
a = b;
b = temp;</code></pre></blockquote>
