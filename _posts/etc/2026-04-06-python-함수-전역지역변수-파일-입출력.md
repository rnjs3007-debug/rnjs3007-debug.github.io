<h2 id="1-계산기-함수-만들기">1. 계산기 함수 만들기</h2>
<p>오늘 첫 번째로 배운 건 함수로 계산기를 만드는 것이었다!  
<code>op</code> 라는 매개변수로 연산자를 받아서 <code>if/elif</code> 로 나눠지는 구조이다.</p>
<pre><code class="language-python">def calc(v1, v2, op):
    result = 0
    if op == '+':
        result = v1 + v2
    elif op == '-':
        result = v1 - v2
    elif op == '*':
        result = v1 * v2
    elif op == '/':
        result = v1 / v2
    return result

oper = input(&quot;계산을 입력하세요(+,-,*,/):&quot;)   # 기본으로 string으로 받음
var1 = int(input(&quot;첫번째 수를 입력하세요:&quot;))   # 문자를 숫자로 형변환
var2 = int(input(&quot;두번째 수를 입력하세요:&quot;))

res = calc(var1, var2, oper)
print(&quot;##계산기 : %d %s %d = %d&quot; % (var1, oper, var2, res))</code></pre>
<blockquote>
<p>💡 <code>input()</code> 은 항상 문자열로 받아온다. 숫자로 쓰려면 <code>int()</code> 로 형변환 필수!</p>
</blockquote>

<h2 id="2-0으로-나누기-예외처리">2. 0으로 나누기 예외처리</h2>
<p>나누기를 할 때 두 번째 수가 0이면 오류가 난다.<br />함수를 호출하기 전에 조건문으로 미리 막아주는 방식을 배웠다.
초반에 &quot;0으로 나누면 안됩니다ㅠㅠ&quot;라는 출력물은 나왔지만, 나눗셈 '0'에 대한 오류가 떠서 살짝 고민했는데, else를 추가하니 문제없이 출력이 가능했다.</p>
<pre><code class="language-python">def calc(v1, v2, op):
    result = 0
    if op == '+':
        result = v1 + v2
    elif op == '-':
        result = v1 - v2
    elif op == '*':
        result = v1 * v2
    elif op == '/':
        result = v1 / v2
    elif op == '**':
        result = v1 ** v2
    return result

var1 = int(input(&quot;첫번째 수를 입력하세요:&quot;))
oper = input(&quot;계산을 입력하세요(+,-,*,/,**):&quot;)
var2 = int(input(&quot;두번째 수를 입력하세요:&quot;))

if oper == '/' and var2 == 0:
    print(&quot;0으로는 나누면 안됩니다ㅠㅠ&quot;)
else:
    res = calc(var1, var2, oper)
    print(&quot;##계산기 : %d %s %d = %d&quot; % (var1, oper, var2, res))</code></pre>
<blockquote>
<p>⚠️ 함수 안에서 막는 방법도 있지만, 호출 전에 조건을 확인하는 것이 더 안전하다.</p>
</blockquote>

<h2 id="3-전역변수-vs-지역변수">3. 전역변수 vs 지역변수</h2>
<p>오늘 가장 헷갈렸던 부분이다.<br />함수 안에서 선언한 변수는 <strong>지역변수</strong>, 함수 밖에서 선언한 변수는 <strong>전역변수</strong>다.</p>
<pre><code class="language-python">def func1():
    a = 10               # 지역변수 — func1 안에서만 존재
    print(&quot;func1()에서 a값 %d&quot; % a)

def func2():
    print(&quot;func2()에서 a값 %d&quot; % a)   # 전역변수 a를 불러옴

a = 20                   # 전역변수 선언

func1()                  # 10 출력
func2()                  # 20 출력</code></pre>
<blockquote>
<p>💡 핵심 순서: <strong>함수 먼저 선언 → 전역변수 선언 → 함수 호출</strong><br />일반적으로 제일 많이 작성하는 순서인것 같다.</p>
</blockquote>
<h3 id="전역변수지역변수-구분-예제">전역변수/지역변수 구분 예제</h3>
<pre><code class="language-python">def func1():
    result = 100
    return result

def func2():
    print(&quot;반환값이 없는 함수 실행&quot;)

hap = 0             # 전역변수 선언

hap = func1()       # 100을 돌려받는다
print(&quot;func1()에서 돌려준 값 ==&gt; %d&quot; % hap)
func2()</code></pre>

<h2 id="4-반환값이-여러-개인-함수">4. 반환값이 여러 개인 함수</h2>
<p>파이썬 함수는 값을 하나만 돌려줄 수 있다.<br />여러 개를 돌려주고 싶으면 <strong>리스트에 담아서 반환</strong>하면 된다.</p>
<pre><code class="language-python">def multi(v1, v2):
    retList = []          # 결과를 담을 리스트 (지역변수)
    res1 = v1 + v2
    res2 = v1 - v2
    retList.append(res1)
    retList.append(res2)
    return retList

myList = []           # 전역변수 선언
hap, sub = 0, 0

myList = multi(100, 200)
hap = myList[0]       # 0번 인덱스에 저장되어있는 값'300'
sub = myList[1]       # 1번 인덱스에 저장되어있는 값'-100'
print(&quot;multi()에서 돌려준 값 ==&gt; %d, %d&quot; % (hap, sub))</code></pre>
<blockquote>
<p>💡 값 여러 개를 받고 싶으면 리스트에 담아서 <code>return</code> 한다.</p>
</blockquote>

<h2 id="5-로또-추첨기">5. 로또 추첨기</h2>
<p><code>random</code> 모듈을 <code>import</code> 해서 1~45 사이의 숫자를 중복 없이 6개 뽑는 프로그램이다.<br /><code>while True</code> 와 <code>break</code> 를 활용한 무한루프 탈출 패턴을 익혔다.</p>
<pre><code class="language-python">import random   # 랜덤 기능을 가져다 쓸 수 있으므로, 초반에 설정한다.

def getNumber():
    return random.randrange(1, 46)   # 1~45 사이 랜덤 숫자

lotto = []
num = 0

print(&quot;** 로또 추첨을 시작합니다. **\n&quot;)

while True:
    num = getNumber()

    if lotto.count(num) == 0:   # 중복 체크 — 리스트에 없을 때만 추가
        lotto.append(num)

    if len(lotto) &gt;= 6:
        break                   # 6개 채우면 탈출

lotto.sort()   # 오름차순 정렬
for i in range(0, 6):
    print(&quot;%d&quot; % lotto[i], end=' ')</code></pre>
<blockquote>
<p>💡 <code>lotto.count(num) == 0</code> → 리스트 안에 <code>num</code> 이 없을 때만 추가, 중복 방지를 위한 count함수를 사용한다.
<img alt="" src="https://velog.velcdn.com/images/addung/post/c218b637-8382-425c-811f-c8425cd47672/image.png" /></p>
</blockquote>

<h2 id="6-파일-입출력">6. 파일 입출력</h2>
<p>파이썬으로 파일을 읽고 쓰는 방법을 배웠다.<br />절대경로와 상대경로의 차이도 중요함에 교수님께서 재차 강조하신 부분.</p>
<ul>
<li><strong>절대경로</strong>: 파일의 전체 경로를 정확하게 지정 (예: <code>C:/doit/newfile.txt</code>)</li>
<li><strong>상대경로</strong>: 현재 실행 중인 파일 기준으로 경로 지정</li>
<li>실제 회사 등에서의 프로그램을 납품할 때는 <strong>상대경로</strong>를 사용해야 한다. 사람마다 파일 경로가 다르기 때문이다.</li>
</ul>
<h3 id="파일-쓰기-w-모드">파일 쓰기 (w 모드)</h3>
<pre><code class="language-python">f = open(&quot;C:/doit/newfile.txt&quot;, 'w')   # w: 쓰기모드, 절대경로
for i in range(1, 11):
    data = &quot;%d번째 줄입니다.\n&quot; % i
    f.write(data)
f.close()   # 반드시 닫아줘야 한다!</code></pre>
<blockquote>
<p>c파일의 doit폴더 중 newfile.txt에서 쓰기모드로 접근한다.</p>
</blockquote>
<h3 id="파일-읽기-r-모드">파일 읽기 (r 모드)</h3>
<pre><code class="language-python"># 한 줄만 읽기
f = open(&quot;C:/doit/newfile.txt&quot;, 'r')
line = f.readline()
print(line)
f.close()

# 전체 읽기
f = open(&quot;C:/doit/newfile.txt&quot;, 'r')
while True:
    line = f.readline()
    if not line: break    # 더 이상 읽을 줄이 없으면 탈출
    print(line)
f.close()</code></pre>

<h2 id="오늘-수업-내용-정리">오늘 수업 내용 정리.</h2>
<ul>
<li><strong>함수선언 → 전역변수 → 호출</strong> 순서로 작성한다</li>
<li>지역변수는 함수 안, 전역변수는 함수 밖에서 선언한다</li>
<li>반환값이 여러 개면 <strong>리스트에 담아서 return</strong> 한다</li>
<li><code>while True</code> + <code>break</code> 조합으로 무한루프를 제어한다</li>
<li>파일은 열면 반드시 <strong>닫아야 한다</strong> (<code>close()</code>)</li>
<li>납품할 때는 절대경로 말고 <strong>상대경로</strong>를 사용한다</li>
</ul>
