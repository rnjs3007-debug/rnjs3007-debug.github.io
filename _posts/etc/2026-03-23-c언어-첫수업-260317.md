<h1 id="1-컴퓨터-구조-및-운영체제">1. 컴퓨터 구조 및 운영체제</h1>
<p>-컴퓨터 하드웨어 구성: 프로세서(CPU), 메모리, 디스플레이, 하드디스크가 시스템 버스로 연결되어 있다.</p>
<p>-기억장치의 특징: 메모리(주기억장치): RAM은 속도가 빠르지만 휘발성 데이터(0, 1의 비트/바이트 단위)를 저장 </p>
<p>-하드디스크(보조기억장치): 비휘발성으로 데이터를 영구 저장하여 유실을 방지</p>
<p>-운영체제: CPU나 메모리 같은 자원을 효율적으로 관리하고 사용자가 컴퓨터를 쉽게 쓸 수 있게 돕는 소프트웨어</p>
<h1 id="2-프로그래밍의-이해와-과정">2. 프로그래밍의 이해와 과정</h1>
<p>-프로그래밍: 컴퓨터를 이용해 문제를 해결하는 과정이며, 문법 습득보다 문제해결력이 더 중요하다.  </p>
<p>AI(GPT 등)는 보조 수단으로만 활용하고, 스스로 손으로 코딩하며 고민해 볼 것을 권장한다!  (교수님의 조언!!) </p>
<h4 id="-개발-프로세스폭포수-모델">-개발 프로세스(폭포수 모델):</h4>
<p>1) 요구사항 명세: 클라이언트와 조율하여 필요한 사항 정의</p>
<p>2) 설계: 모듈별로 문서화(Figma, UML 등 활용) </p>
<p>3) 개발(코딩): 설계 문서를 기반으로 실제 코드 작성 </p>
<p>4) 테스트: 화이트박스/블랙박스 테스트 수행 </p>
<p>5) 검수 및 수정: 개발/설계 단계로 돌아가 오류 수정 후 최종 인수 테스트 거쳐 완성 </p>
<h1 id="3--프로그래밍-언어의-특징">3.  프로그래밍 언어의 특징</h1>
<p>-언어의 종류: 고급 언어: C, Java, Python 등 사람이 이해하기 쉬운 언어</p>
<p>-어셈블리어/기계어: 컴퓨터가 직접 실행할 수 있는 저급 언어</p>
<h4 id="-실행-방식">-실행 방식:</h4>
<p>컴파일(C언어): 소스 코드를 실행 파일로 변환하여 실행 속도가 빠름 </p>
<p>인터프리터(Python): 한 줄씩 번역하며 실행하여 느리지만 호환성이 좋음</p>
<p>개발 환경: Visual Studio 같은 통합 개발 환경(IDE)을 사용하며, 한 프로젝트 내 파일에는 main 함수가 하나만 존재해야 함</p>
<h1 id="4-c언어-기본-문법-및-자료형">4. C언어 기본 문법 및 자료형</h1>
<p>-변수(Variable): 데이터를 저장하는 공간으로, 선언 시 메모리 공간이 확보된다.</p>
<p>-명명 규칙: 숫자로 시작 불가, 특수문자는 언더바(_)만 허용, 대소문자 구분 필수</p>
<p>-표기법: 단어를 구분하는 카멜 표기법(예: areaCircle) 등을 사용</p>
<h4 id="-자료형-상세">-자료형 상세:</h4>
<p>-signed (기본값): 음수와 양수 모두 표현 (예: -128 ~ 127)</p>
<p>-unsigned: 음수를 포기하는 대신 양수의 범위를 2배로 넓힘 (예: 0 ~ 255)</p>
<p>*문자
char: 1바이트(8비트) (11111111/256개)
signed -128<del>127
unsigned 0</del>255 (양수 한해서만 해당)</p>
<p>*정수형
short: 2바이트(16bit) -65536개
int: 4바이트(32bit), %d 사용
signed -32768<del>32767
unsigned 0</del>65535</p>
<p>*실수형(소수점이 있는 숫자들)
float : 4바이트, %f 사용
double : 8바이트(64bit) ,  %1f 사용/ 모르면 일단 더블 쓰자!
double latitude;(위도)
double longitue;(경도)</p>
<h4 id="핵심-문법">핵심 문법:</h4>
<p>문장 끝에는 반드시 세미콜론(;) 사용 </p>
<p>-scanf: 사용자로부터 값을 입력받는 함수이며, 안전한 사용을 위해 
'#pragma warning(disable:4996)' 설정이 필요할 수 있다.</p>
<p>-자동 형 변환: 컴파일러가 상황에 맞춰 자료형을 변환한다 (예: int n = 2.3; 작성 시 정수부에 해당하는 '2'만 저장).</p>
<h1 id="5-c언어-실습하기">5. C언어 실습하기</h1>
<p>{*C언어 소스 파일(.c) 추가하기 (매우 중요!)</p>
<p>1)오른쪽 [솔루션 탐색기] 창에서 [소스 파일] 폴더를 마우스 오른쪽 버튼으로 클릭</p>
<p>2)[추가] -&gt; [새 항목]을 선택합니다.</p>
<p>중간에 이름을 적는 칸에서 소스.cpp를 지우고 main.c라고 입력 (뒤에 .c를 꼭 직접 써주기!)</p>
<p>[추가] 버튼을 누름&gt; 실행가능}</p>

<pre><code>#include &lt;stdio.h&gt;

int main(void)

<p>오류 방지 코드 작성필요함.(현업에서는 잘 사용하지는 않음)

<pre><code>#include &lt;stdio.h&gt;

#pragma warning(disable:4996)

int main(void)

{
    int a;
    printf(&quot;input number: &quot;);
    scanf(&quot;%d&quot;, &amp;a);

    printf(&quot;a: %d\n&quot;, a);
    return 0;

}</code></pre><blockquote>
<p>c언어 에서는, %d와 같은 변수형태를 지정하고, 이후에 입력한 값의 이름을 불러온다.</p>
</blockquote>
<p>*원의 넓이 구하는 문제 </p>
<pre><code>#include &lt;stdio.h&gt;

#pragma warning(disable:4996)

int main(void)

{
    double PI = 3.141592653;
    double result;
    float radius;   /*변수를 먼저 선언함, 변수형태 다 작성*/

    printf(&quot;input radius: &quot;);
    scanf(&quot;%f&quot;, &amp;radius);

    result = radius * radius * PI;

    printf(&quot;area of circle : %f\n&quot;, result);
    return 0;   
}</code></pre><blockquote>
<ol>
<li>먼저 변수형을 설정한 변수를 선언한다.</li>
<li>입력값을 먼저 출력한다. (input 받을값)</li>
<li>scanf로 받는다</li>
<li>입력값을 반영한 새로운 변수를 선언한다.</li>
<li>출력한다./ 단 항상 printf안에 &quot;&quot;입력 하고, 맨 마지막에는 ; 닫는다. 필히!!</li>
</ol>
</blockquote>
